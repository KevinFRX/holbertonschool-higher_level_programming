The ``1-my_list`` module
========================

Using ``MyList``
----------------

Import:

	>>> MyList = __import__('1-my_list').MyList

Use:

	>>> my_list = MyList()

NORMAL

	>>> my_list = MyList([3, 2, 1])

	>>> my_list.print_sorted()
	[1, 2, 3]

EMPTY AND APPEND

	>>> my_list = MyList()

	>>> my_list.print_sorted()
	[]

	>>> my_list.append(4)

	>>> my_list.append(3)

	>>> my_list.append(2)

	>>> my_list.append(1)

	>>> my_list.print_sorted()
	[1, 2, 3, 4]

STRING

	>>> my_list = MyList("Hola")

	>>> my_list.print_sorted()
	['H', 'a', 'l', 'o']
